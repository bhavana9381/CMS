Key Stakeholders

Entities Involved in Personal Data Processing and Consent Management

Consent Management System (CMS)

Data Principal

The individual whose personal data is being collected and processed.

Data Processor

Processes personal data on behalf of the Data Fiduciary securely.

DPO

Data Protection Officer ensures organizational compliance with DPDP rules.

Consent Manager

Platform to manage, review, and withdraw consent in a centralized manner.

Data Fiduciary

Entity that determines the purpose and means of personal data processing.

Stakeholder Responsibilities
Data Principal
Grants or Withdraws Consent
Data Fiduciary
Collects and Uses Data
Data Processor
Processes Data Securely
Consent Manager
Manages Consent Lifecycle
DPO
Ensures Regulatory Compliance
